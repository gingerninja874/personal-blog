Wednesday 29th April 2020 - The New Regime

06:30 - Wake up, shower, fuel the brain with coffee.
07:00 - Start 1 hour coding block.
08:00 - 10 minute break.
08:10 - Start 1 hour coding block.
09:10 - 10 minute break.
09:20 - Start 1 hour coding block.
10:20 - Rest of day goes as follows:
  for (fatigue = arbitrary; fatigue < brainpower; fatigue++) {
      Start another hour of coding;
  }
Otherwise, rest. You deserve it

This is the current plan for my coding going forward. It takes inspiration Code Drip's video "How I stayed focused while learning to code" in which he suggests to code in the morning and to play around with your morning schedule. The second inspiration was the reddit post "Spent the last 100 days learning to code, and told no one." which was extremely inspirational and sets out an awesome framework for what can be done. I'm following his 3 hours a day of 1 hour blocks, as well as using forest app on my phone stop any distractions whilst coding.
